Virtual Mouse

The Virtual Mouse project is a Python-based application that allows users to control their computer's cursor using hand gestures. Leveraging OpenCV for real-time computer vision and MediaPipe for hand detection, the system interprets hand movements and converts them into mouse actions. Features include hand detection, gesture recognition, cursor movement, and customizable settings. The project aims to provide an accessible and hygienic alternative to traditional input devices, useful in shared environments and for individuals with mobility impairments. Future enhancements may include improved gesture recognition, multi-hand support, and machine learning integration for greater accuracy and responsiveness..
